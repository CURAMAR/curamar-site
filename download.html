Discord で配布を受け取る

新しいバージョンへの差し替え方法（v0.6.0 から「フォルダ」配布）

v0.6.0 から Ligastmir はフォルダ形式（onedir）での配布になりました。更新は「フォルダを新しいものにして、設定ファイルだけ引き継ぐ」流れです。設定は別ファイルなので、コピーすれば引き継がれます。

設定はコピーで引き継げます。
設定・読み上げ辞書・視聴者メモ・着せ替え素材は、Ligastmir.exe とは別のファイル／フォルダに保存されています。 新しいフォルダへ下記をコピーすれば、設定はそのまま引き継がれます。
新しい Ligastmir をダウンロードして展開する
最新版の zip を入手し、解凍して新しい Ligastmir_v0.6.0… フォルダを取り出します。中に Ligastmir.exe と _internal フォルダが入っています。
今使っている Ligastmir を閉じる
起動中だとファイルを移せないので、Ligastmir を終了します。
古いフォルダから設定ファイルをコピーする
今まで使っていたフォルダ（旧 StreamCore.exe / 旧 Ligastmir.exe があった場所）から、次のファイル・フォルダを新しいフォルダ（Ligastmir.exe と同じ階層）へコピーします。
config.json / tts_dict.json / viewers.json / viewer_stats.json / user_assets フォルダ
※ _internal フォルダはプログラム本体なので触らないでください。
起動して完了
新しいフォルダの Ligastmir.exe（または同じ場所の 起動.bat）から起動します。設定が引き継がれた状態で立ち上がれば完了です。古いフォルダはしばらくバックアップとして残しておくと安心です。
StreamCore から Ligastmir に移行する方へ：名前とフォルダ構成が変わりましたが、上の手順で設定（config.json など）をコピーすれば、これまでの設定をそのまま引き継げます。うまく起動しない・設定が消えたように見える場合は、コピー先が Ligastmir.exe と同じ階層になっているかを確認のうえ、下のサポートへご連絡ください。

困った時・報告したい時

不具合の報告や要望は、以下のどちらからでも受け付けています。

COMMUNITY（おすすめ）

Discord で伝える

その場でやり取りできるので解決が早いです。ログファイルもチャンネルに添付できます。

Discord に参加
FORM（Discord なしでOK）

フォームで送る

Discord を使わない方はこちら。ログファイルの添付もできます。

報告フォームを開く

ログファイルの送り方

不具合の調査には、ログファイルがとても役立ちます。送る時は次のようにしてください。

ログファイルを見つける
アプリの「INFO」ページの「ログフォルダを開く」から、ログのある場所を開けます。
送る
サイズが小さいログは、Discord の #ログ共有 にそのまま添付するか、報告フォームに添付してください。
サイズが大きいログは、ギガファイル便などのファイル転送サービスにアップロードして、その URL を Discord か報告フォームに貼ってください。
