海外コメントを日本語に翻訳して読み上げる方法

Ligastmir（無料）を使うと、海外視聴者のコメントが原文＋日本語訳の2段表示になり、 日本語訳が読み上げられます。言語は自動検出——英語・中国語・韓国語・スペイン語など、 翻訳エンジンが対応する言語はすべて対象。逆方向に、自分の字幕へ英訳を重ねることもできます。 APIキーなしの無料エンジンで動き、DeepL キーがあれば高品質になります。

できあがりの例

最短手順（3ステップ）

Ligastmir を起動して配信に接続する起動.bat から起動し、「配信」ページで Twitch のチャンネル名を入れて接続（匿名でOK）。YouTube は配信URLとAPIキーで接続します。
「チャット翻訳」をONにする「音声・コメント」ページの「💬コメント関連」にあります。スイッチ1つだけです。
外国語コメントが自動で翻訳されるコメントビューに原文＋日本語訳の2段表示、読み上げは日本語訳。どの言語かを設定する必要はありません（自動検出）。

翻訳エンジンについて

  • 無料エンジンが標準 — APIキー登録なしでそのまま動きます。一時的に不調でも別エンジンへ自動フォールバックします。
  • DeepL キー対応（任意） — 無料の DeepL API キーを設定すると、より自然な訳になります。
  • 読み上げ辞書と共存 — 辞書登録した固有名詞は、翻訳を通しても読み方が保たれます。

うまくいかないとき

翻訳されない 「チャット翻訳」がOFFのことがほとんどです。「音声・コメント」ページで確認してください。→ 使い方 ⑤コメント連動・翻訳
短い日本語コメントが誤って翻訳される・スタンプが変に訳される 判定は自動調整されており、スタンプ（エモート）のコードは翻訳対象から除外されます。気になる例があれば Discord で教えてください。
