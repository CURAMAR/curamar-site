PRIVACY POLICY

プライバシーポリシー

最終更新: 2026年7月3日
本ポリシーは、curamar が運営するウェブサイト curamar.space と、配信支援ツール Ligastmir（以下「本ソフト」）における、情報の取り扱いについて説明するものです。 本ソフトは「あなたの個人情報をできるだけ集めない・端末の外に出さない」という方針で設計されています。

1. Ligastmir（本ソフト）について

1-1. 本ソフトが収集しないもの

本ソフトは、利用者や視聴者の個人を特定する情報を収集・蓄積しません。具体的には：

  • 視聴者の名前やコメントを、個人を識別できる形では記録しません。ログ機能は統計（件数・処理時間など）が中心で、視聴者個人を特定する情報は残さない設計です。
  • 利用者の氏名・住所・連絡先などを、本ソフトが収集してどこかに送ることはありません。
  • 利用者の行動を追跡したり、プロファイリングのためにデータを集めることはありません。

1-2. 端末内にのみ保存される情報

以下の情報は、すべてあなたのパソコンの中だけに保存され、開発者を含む第三者が見ることはできません：

  • 各種設定（config）、読み上げ辞書、視聴者メモ、着せ替え素材、動作ログ
  • 連携サービスの API キーや認証トークン。これらはお使いのパソコン固有の情報から生成した鍵で暗号化して保存されます。別のパソコンに移すと復号できないため、移行時は再入力が必要です。

1-3. 外部サービスとの連携で送信される情報

本ソフトは、利用者が選んで有効にした機能に限り、以下の外部サービスと通信します。どの連携も、利用者が明示的にオンにしたときだけ動作します（既定はオフ）。

翻訳機能について（重要）
翻訳機能を有効にすると、翻訳の対象となるテキスト（コメントなど）が、選択した外部の翻訳サービスに送信されます。 利用しうるサービスは DeepL、Google 翻訳（deep-translator 経由）、MyMemory、LibreTranslate です。 送信されるのは翻訳対象の文章であり、各サービスのプライバシーポリシー・利用規約が適用されます。 翻訳機能をオフにすれば、これらへの送信は行われません。
  • Twitch / YouTube: チャットの取得や配信情報（タイトル・カテゴリなど）の連携のため、各社の API と通信します（利用者が連携を有効にした場合）。
  • VOICEVOX / 棒読みちゃん: 読み上げ機能のため、お使いのパソコン上で動作するこれらのソフトに、読み上げるテキストを渡します（ローカル通信）。
  • わんコメ（OneComme）: 同伴起動の機能を使う場合、お使いのパソコンにインストール済みのわんコメ本体を起動・終了します。わんコメへデータを送ったり、コメントを取り込む連携は行いません（起動/終了のみ）。
  • OBS: 配信ソフトとの連携のため、お使いのパソコン上の OBS と通信します（ローカル通信）。
  • 更新のお知らせ: 「更新を確認する」を有効にした場合に限り、最新版の情報（version.json）を curamar.space から取得します。既定はオフで、オフの間は一切通信しません。

1-4. YouTube API サービスの利用について

本ソフトは YouTube API Services を利用しています。本ソフトの YouTube 連携機能を利用する場合、利用者は YouTube の利用規約 に同意したものとみなされ、また Google プライバシーポリシー が適用されます。本ソフトは、YouTube から取得した情報を、上記の連携機能を提供する目的以外には利用しません。

本ソフトに与えた YouTube（Google アカウント）へのアクセス権限は、 Google アカウントのセキュリティ設定 からいつでも取り消すことができます。本ソフト内の「ログアウト」からも認証情報を削除できます。

2. ウェブサイト（curamar.space）について

2-1. アクセス解析

本サイトでは、訪問者数やどのページが見られているかを把握するために、アクセス解析を利用しています。 これは本サイトの改善のために、閲覧の傾向を統計的に把握する目的で用いるものです。 個人を特定する目的では利用しません。

2-2. 外部リンク

本サイトには Twitch、カクヨム、Discord などの外部サービスへのリンクが含まれます。 リンク先での情報の取り扱いについては、各サービスのプライバシーポリシーをご確認ください。

3. お問い合わせ

本ポリシーや情報の取り扱いについてのご質問は、Discord またはサイト上の連絡先からお寄せください。

4. 改定について

本ポリシーの内容は、機能の追加やサービスの変更に応じて改定することがあります。 重要な変更がある場合は、本ページ上で告知します。

本ポリシーは、Ligastmir および curamar.space の現在の仕様に基づいて記載しています。 個人開発のプロジェクトであり、法務の専門家による作成ではありません。記載に不足や誤りを見つけられた場合は、お知らせいただけると助かります。
