3分ではじめる Ligastmir

むずかしい設定はあとまわしで OK。まずは動かして、雰囲気をつかむための最短手順です。 詳しい設定やトラブル対処は 使い方ガイド にすべてまとまっているので、 このページでは「とりあえず使ってみる」だけを目指します。

最短セットアップ（5ステップ）

最新版をダウンロードダウンロードページから zip を入手します（無料・登録不要）。Discord で受け取ることもできます。
zip を解凍するzip を右クリック →「すべて展開」。出てきたフォルダは好きな場所に置いて OK です（デスクトップでも可）。
起動.bat をダブルクリックフォルダの中の 起動.bat から起動します。Ligastmir.exe を直接起動することもできますが、ダウンロード時のブロック解除を自動で行ってくれるので、通常は 起動.bat がおすすめです。初回に SmartScreen の画面が出たら「詳細情報 → 実行」で進めます（公式サイト・公式 Discord から入手した配布版であれば大丈夫です）。
使いたい機能だけ ON にする最初の画面の案内に沿って、使いたい機能だけスイッチを入れます。字幕だけ・読み上げだけでも「正解」です。Twitch / YouTube / OBS の接続も、使うものだけでかまいません。
OBS にブラウザソースを追加する画面に出したいもの（字幕・コメントビューなど）の URL がツール内に一覧で用意されています。コピーして OBS の「ブラウザ」ソースに貼れば表示されます。
これで完了です。うまくいかないところがあったら、FAQ か 使い方ガイド を見るか、 Discord で気軽に聞いてください。

「まずはこれだけ」の最小レシピ

全部の機能を覚える必要はありません。目的ひとつだけの最小構成なら、さらに簡単です。

🎤 まず字幕だけ使う

  1. 起動して字幕（音声認識）を ON
  2. 字幕ページの URL をコピー
  3. OBS のブラウザソースに貼る
  4. マイクに向かって話す → 画面に字幕が出ます

🔊 まず読み上げだけ使う

  1. 起動して Twitch か YouTube に接続
  2. 読み上げを ON（Windows 標準の声ならそのまま使えます）
  3. コメントが来ると読み上げてくれます
  4. VOICEVOX の声を使いたくなったらガイドへ
試したら、よければ一言ください。 「ここが分かりづらかった」「この設定で迷った」——そんな一言だけでも、開発のとても大きな助けになります。 Discord でも 匿名フォーム でも OK です。
